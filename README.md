
# Sturla sine config filer

Her kan jeg hente konfigurasjonsfiler for diverse programmer

## Hvordan bruke

1. Klon dette repoet i en mappe, for eksempel ~/dotfiles
2. Installer alle [programmene](./programmer.md)
3. Kjør [Scriptet for å stowe konfigurasjonsmappene](./stow-dotfiles.sh)
4. done

## Annet
stow-dotfiles.sh scriptet må kjøres hver gang man legger til eller fjerner en fil for at endringen skal bli "stowet" riktig
Man behøver ikke kjøre scriptet hvis man kun endrer en fil
